## [1.0.0] - 2026-06-24

### Added

* Initial public release of Flow for Android.
* Local-first expense tracking with categories, payment methods, dates, descriptions, and online/offline transaction context.
* Budget planning with category-based spending limits and monthly progress tracking.
* Finance notes and graph views for reviewing spending context and patterns.
* Account, settings, onboarding, backup, and restore flows for Android use.
* Import/export support using user-controlled local JSON backup files.
* Android packaging through Capacitor with native support for reminders, backup handling, and system bar behavior.
* GitHub repository foundation with verification workflow, issue templates, pull request template, public documentation, privacy policy, and security policy.
