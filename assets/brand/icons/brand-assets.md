# Brand Assets

This directory contains the original source branding assets for Flow.

## Source Icon Files

- `flow_icon_preview_1024.png`: Visual mockup of the composite launcher icon.
- `flow_adaptive_icon_background_1024.png`: Base layer color asset for Android adaptive launcher icon.
- `flow_adaptive_icon_foreground_1024.png`: Logo layer design with custom scale and positioning balance.
- `flow_monochrome_icon_1024.png`: Dark theme launcher design for Android monochrome theme.
